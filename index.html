Welcome to the Redneck IT RPG Page

How do I participate in games?
  1. Create a free account on Roll20.net.
  2. Use your Redneck IT domain login to access Redneck IT RemoteApps.
  3. Create a character using one of the following character generators:
    • Hero Labs: Used to create characters for DnD 3.5e, DnD 5e, and Pathfinder
    • GURPS Character Assistant: Used to create any GURPS Character
    • Dresden RPG Character Manager: Used to create any Dresden Files (Fate System) character
  4. Email gaming@redneckit.net and include the following:
    • Name
    • Email you used to create your Roll20.net account.
    • Character Name
  5. Watch some videos!
    • How do you play an RPG?
    • How do you use Roll20?
    • How do you use Hero Lab?

Standard Array of Ability Scores:
15, 14, 13, 12, 10, 8


What are the ability scores, and what do they mean?
Strength is being able to crush a tomato.
Dexterity is being able to dodge a tomato.
Constitution is being able to eat a bad tomato.
Intelligence is knowing a tomato is a fruit.
Wisdom is knowing not to put a tomato in a fruit salad.
Charisma is being able to sell a tomato based fruit salad.
