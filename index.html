How do I participate in games?
  1. Create a free account on Roll20.net.
  2. Use your Redneck IT domain login to access Redneck IT RemoteApps
  3. Create a character using one of the character generators.
    • Hero Labs: DnD 3.5e, DnD 5e, Pathfinder,
    • GURPS Character Assistant: Any GURPS Character
    • Dresden RPG Character Manager: Any Dresden Files Fate Character
  4. Learn to use Roll20.
